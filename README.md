# Getting Started:
#### Clone the repository
`git clone https://github.com/codeprofile/django-todo-app.git`

#### Navigate to the project directory
`cd django-todo-app`

#### Install dependencies
`pip install -r requirements.txt`

#### Run the application
`python manage.py runserver`
